PyOceanMaps
===========

*Uses python - matplotlib - basemap to create maps*

##### Requirements #####
* [ETOPO1 file](http://www.ngdc.noaa.gov/mgg/global/relief/ETOPO1/data/ice_surface/grid_registered/netcdf/ETOPO1_Ice_g_gmt4.grd.gz) 377MB
* Python 2.6 or 2.7
* Matplotlib
* Basemap (mpl_toolkits)

##### Package Contents #####
* this readme
* bath_map
* bath_section
* haversine
